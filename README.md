# Resume Refiner
Upload a resume with the styles you like and yours (in its current form), output an updated resume in the reference style you like! 

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
